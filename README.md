# GuiBeStore

Sistema de vendas feito com PHP procedural, HTML, CSS e MySQL.

## Como executar

1. Ligue Apache e MySQL no XAMPP.
2. Importe `banco.sql` no phpMyAdmin.
3. Abra `http://localhost/PHP/SiteDeVendas/`.
